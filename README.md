🚀 Completed the "Build With Databricks: Hands-On Project Challenge" by Codebasics.

I built an end-to-end Data & AI pipeline using Databricks to identify high-value customers from an e-commerce dataset.

🔹 Project Highlights

• Implemented Medallion Architecture (Bronze → Silver → Gold)
• Built data pipelines using PySpark and Delta tables
• Engineered customer behavior features from transaction data
• Trained a Random Forest model to classify high-value customers
• Achieved strong model accuracy and tracked experiments using MLflow

📊 Technologies Used

Databricks | PySpark | Delta Lake | Python | Scikit-learn | MLflow

This project helped me understand how modern data platforms combine **data engineering, analytics, and machine learning into a unified pipeline**.
